# Healthcare Compliance Audit Workflow

Comprehensive healthcare compliance validation using compliance and security agents:

1. **compliance-officer**: Conducts HIPAA and regulatory compliance assessment
2. **security-engineer**: Performs security audit and vulnerability assessment
3. **accessibility-tester**: Validates accessibility compliance for healthcare users
4. **ui-auditor**: Reviews UI compliance with healthcare design standards

## Usage
Use this command for regular compliance audits or before major healthcare feature releases.

## Process  
The command coordinates compliance agents to:
- Validate comprehensive HIPAA and healthcare regulatory compliance
- Conduct security vulnerability assessment and risk analysis
- Ensure accessibility compliance for diverse healthcare users
- Review UI compliance with healthcare industry standards

## Output
Comprehensive compliance audit report with regulatory validation, security assessment, and remediation recommendations.
