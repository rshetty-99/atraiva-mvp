# Full Testing Workflow

Executes a comprehensive testing suite using all testing agents:

1. **test-engineer**: Creates overall testing strategy and test plans
2. **e2e-tester**: Implements end-to-end healthcare workflow testing
3. **accessibility-tester**: Validates WCAG compliance and inclusive design
4. **performance-tester**: Conducts load testing and performance validation

## Usage
Use this command before major releases or when comprehensive testing validation is required.

## Process
The command coordinates testing agents to provide:
- Complete test strategy development
- End-to-end healthcare workflow validation
- Accessibility and compliance testing
- Performance and scalability testing
- Security and vulnerability assessment

## Output
Comprehensive testing report with quality metrics, compliance validation, and performance benchmarks.
