# Sprint Planning Workflow

Activates the complete project management suite for sprint planning:

1. **sprint-coordinator**: Facilitates sprint planning and agile ceremonies
2. **task-orchestrator**: Breaks down complex healthcare projects into manageable tasks
3. **delivery-manager**: Plans releases and coordinates delivery timelines
4. **product-strategist**: Provides strategic context for feature prioritization

## Usage
Use this command when planning new development sprints or major healthcare feature releases.

## Process
The command coordinates project management agents to:
- Plan sprint cycles with healthcare compliance considerations
- Break down complex healthcare features into development tasks
- Coordinate delivery timelines with business objectives
- Prioritize features based on healthcare user value and strategic goals

## Output
Comprehensive sprint plan with task breakdown, timeline estimates, and delivery coordination.
