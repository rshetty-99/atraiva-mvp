# Design Review Workflow

Triggers a comprehensive design review process using multiple specialized agents:

1. **ui-designer**: Analyzes and implements design requirements
2. **ui-auditor**: Conducts thorough design quality assurance 
3. **ux-researcher**: Validates user experience and accessibility
4. **accessibility-tester**: Ensures WCAG 2.1 AA compliance

## Usage
Use this command when you need a complete design review cycle for any UI component or feature.

## Process
The command automatically coordinates between design agents to provide:
- Design implementation review
- Accessibility compliance validation  
- User experience assessment
- Performance impact analysis
- Healthcare compliance verification

## Output
Comprehensive design review report with actionable recommendations and compliance validation.
